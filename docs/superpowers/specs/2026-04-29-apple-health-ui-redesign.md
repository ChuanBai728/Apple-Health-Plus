# Apple Health+ 前端 UI 重设计

## 概述

将 Apple Health+ 前端从传统 Tailwind 灰白卡片风格重新设计为** iOS 原生风格**，参考 Apple Fitness/Health 应用的视觉语言，实现多彩活动环配色 + 全浅色基调 + 毛玻璃效果。

- **日期**: 2026-04-29
- **设计方向**: iOS 原生风格
- **基调**: 全浅色（iOS systemGroupedBackground #F2F2F7）
- **配色**: 多彩活动环（红/橙/绿/紫/蓝按健康类别分色）

---

## 设计 Tokens

### 色彩

| Token | 值 | 用途 |
|--------|-----|------|
| 背景色 | `#F2F2F7` | 全页背景 |
| 卡片色 | `rgba(255,255,255,0.72)` + `backdrop-blur: 20px` | 毛玻璃卡片 |
| Hero 渐变 | `#FFFFFF → #F8F8FC → #EEF0FF` | Hero 区域白→浅蓝紫 |
| AI 叙述渐变 | `#FAFAFE → #F4F4FC` | AI 内容区域 |
| 标题文字 | `#1C1C1E` | 主要标题 |
| 辅助文字 | `#8E8E93` | 标签、说明 |
| 正文文字 | `#3A3A3C` | 段落文字 |

### 强调色（按健康类别）

| 类别 | 颜色 | Hex |
|------|------|-----|
| 活动/心率 | 红色 | `#FF3B30` |
| 能量/运动 | 橙色 | `#FF9500` |
| 身体/体重 | 绿色 | `#34C759` |
| 睡眠/恢复 | 紫色 | `#5856D6` |
| 通用/链接 | 蓝色 | `#007AFF` |

### 形状

| 元素 | 值 |
|------|-----|
| 卡片圆角 | `16px` (rounded-2xl) |
| Hero 圆角 | `24px` |
| 按钮/徽章 | `full` (rounded-full) |
| 阴影 | `0 2px 12px rgba(0,0,0,0.03)` |
| 边框 | `1px solid rgba(0,0,0,0.04)` |

### 字体

- 字体栈: `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif`
- 大标题: `28px / font-weight: 700 / letter-spacing: -0.5px`
- 页面标题: `22px / font-weight: 700`
- 正文: `13px / font-weight: 400`
- 标签: `10px / font-weight: 600 / uppercase / letter-spacing: 0.5px`
- 数值: `font-variant-numeric: tabular-nums`

## 页面设计

### 1. 概览仪表盘 (`/reports/[id]/overview`)

**布局**: Bento Grid (4 列)，最大宽度 960px

**Hero 卡片** (2 列 x 2 行):
- 白→浅蓝紫渐变背景
- 左上：大号步数 (36px) + 进度条
- 右上：三色活动环 (红 Move / 绿 Exercise / 蓝 Stand)，无文字
- 底部三列：心率 | HRV | 睡眠
- 底部链接：AI 健康报告入口

**指标快照卡片** (1x1):
- 毛玻璃白卡片
- 彩色圆点 (按类别) + 标签
- 大号数值 (24px)
- 迷你 SVG 趋势线

**异常告警** (2 列宽):
- 暖黄色背景 `#FFFBF5` + 橙色边框
- 异常数量 + 简短建议

**分段控件**:
- 日/周/月/年切换，iOS 风格药丸形

### 2. AI 健康报告 (`/reports/[id]/report`)

**布局**: 居中窄布局 (max-w-2xl)

**封面**:
- 白→浅蓝紫渐变
- 英文副标题 (WEEKLY REPORT) + 中文大标题
- 日期范围 + 数据天数
- 4 色状态分布标签（高压=红, 恢复=绿, 运动=橙, 平稳=灰）

**核心指标变化**:
- 毛玻璃白卡片
- 每行：彩色圆点 + 指标名 + 数值 + 趋势徽章
- 趋势徽章：涨=绿色背景, 跌=红色背景, 平=灰色背景

**AI 叙述**:
- 浅紫白渐变卡片
- 章节标题使用系统强调色（蓝/绿/橙/紫）
- 底部免责声明

### 3. AI 聊天 (`/reports/[id]/chat`)

**布局**: 居中 (max-w-3xl)

**空状态**:
- 机器人图标 + 提示文字
- 5 个建议问题药丸按钮 (F2F2F7 背景)

**消息气泡**:
- 用户: 蓝色 (#007AFF) + 白色文字, 右对齐, 圆角 16px 右下 4px
- AI: 浅灰 (#E9E9EF) + 深色文字, 左对齐, 圆角 16px 左下 4px

**输入区**:
- 圆角药丸输入框 (border-radius: 20px)
- 蓝色发送按钮

### 4. 指标详情 (`/reports/[id]/metrics/[key]`)

**布局**: 居中 (max-w-3xl)

**健康参考卡片**:
- 蓝白渐变 + 2 列 (健康范围/建议)

**标题区**:
- 大号指标名 (28px) + 单位
- "问 AI" 按钮

**图表**:
- Recharts 折线图, 蓝色线条
- 浅灰虚线网格

**统计卡片** (4 列):
- 白卡片 + 左边框彩色条
- 平均值 / 最低 / 最高 / 最新

### 5. 热力图 (`/reports/[id]/heatmap`)

**布局**: 居中

**指标选择器**:
- 水平药丸按钮 (步数/能量/锻炼等)
- 选中=白底, 未选中=透明

**热力图网格**:
- 密集日历视图 (14px 单元格)
- 绿色强度: 浅绿→深绿
- 图例: 少 → 多

### 6. 上传进度 (`/uploads/[id]`)

**布局**: 居中窄 (max-w-lg)

**阶段追踪**:
- 6 阶段列表 (已上传→排队→解析→解析完成→聚合→就绪)
- 当前阶段: 蓝色高亮 + 脉冲圆点
- 已完成: 绿色勾号

**进度条**:
- 圆角 + 蓝紫渐变填充
- 文件名 + 百分比

**完成状态**:
- 绿色背景 + 庆祝图标
- "查看健康报告" 按钮

### 7. 首页 (`/`)

**布局**: 居中 (max-w-6xl)

**Hero**:
- 蓝色徽章 (AI 驱动) + 大标题 + 副标题

**导出指南**:
- 蓝→靛蓝渐变卡片 + 5 步说明

**上传区**:
- 虚线边框拖拽区 (border-dashed)
- 闲置: 灰边, 悬停: 蓝边
- 渐变上传按钮

**演示按钮**:
- 白底 + 紫色边框

**功能卡片** (3 列):
- 白卡片 + 彩色渐变顶边

## 通用变化

1. `globals.css`: 添加 Apple 系统字体栈, 移除旧 shimmer 动画中的硬编码蓝色
2. `layout.tsx`: 背景从 `bg-gray-50` 改为 `bg-[#F2F2F7]`, 顶部导航栏改为毛玻璃效果
3. `types.ts`: 无需修改（标签/单位/参考数据保持不变）
4. 颜色方案从蓝色单色 → 多彩类别色 (红/橙/绿/紫/蓝)
5. 所有卡片从纯白 `bg-white` + `border-gray-200` → 毛玻璃 `bg-white/70` + `backdrop-blur-xl`
6. 圆角从 `rounded-xl` (12px) → `rounded-2xl` (16px) / `rounded-3xl` (24px)
